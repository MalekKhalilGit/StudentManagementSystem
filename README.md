# StudentManagementSystem
School Management System to manage students
I used Java, ThymeLeaf, MySQL, Bootstrap and Spring Boot to build this application.
With this app you can manage a MySQL database.
Functions are:
-adding new students to the database
-updating students that are already in the system
-deleting students from the system
If you press on the add student or update student button you get rediceted to another page where you can enter the information for the new/existing student.
Every stundent has a first name, last name, email and class.
The students are shown in a table on the main page where you can acess the different functions.
The application is build in a way that makes implementing new features possible.
